# weather-cli-app
A Python CLI app that fetches real-time weather using OpenWeatherMap API with search history and JSON persistence.

## Features
- Fetch real-time weather by city name
- Temperature in Celsius
- Search history with timestamps
- Data saved to JSON file (persistent across sessions)
- Error handling for invalid cities and API failures

## Technologies Used
- Python 3
- OpenWeatherMap API
- JSON (data storage)
- OOP (Object-Oriented Programming)
- requests library

## How to Run
1. Make sure Python 3 is installed
2. Install requests: `pip install requests`
3. Clone or download this repo
4. Add your OpenWeatherMap API key in the code
5. Run: `python weather_checker.py`
